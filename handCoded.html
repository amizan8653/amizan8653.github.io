handCoded¶

This module contains the handCoded agent class.

class handCoded.handCoded(worldRef, simIndex, noisyPos, sigma, agentType, noisyBallPos, maxPlayerSpeed, maxBallSpeed, posession=False)¶

This class is a child of the agent class, and therefore inherits all of agent’s methods. This child class will simply over ride the decisionFunction, which is responsible for making the ballholder decide if it should hold the ball, or pass to a team mate. And if passing to a team mate, which team mate to pass to. The hand coded agent will use a deterministic policy.
