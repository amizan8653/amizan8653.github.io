Welcome to keepaway's documentation!
====================================

This is the documentation to the object oriented version of keepaway!
this program is meant to allow AI algorithms to be implemented on
the keepers to allow them to decide if they want to hold the ball, 
or pass to either team mate. Everything else is deterministic.

Here is a video demonstration of the hand coded agents playing:
https://youtu.be/aEgvQnbLsk4
